## **Projeto**
Uma pagina de captura para divulgar um workshop integrando form com mailchimp utilizando um design clean e responsivo.
Com recebimento de Email pelo "Mailchimp" que cria um lista de emails para o palestrante entrar em contato com seus clientes.

## **Tecnologias**
- HTML
- CSS
